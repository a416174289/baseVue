## 项目结构
### 项目使用UI框架为 vue-cli4.x + vuex + iview&elementUI + mock + express
### ###########目录结构描述
### ├── Readme.md                   // 说明
### ├── app                         // 应用
### ├── mock                        // MOCK数据
### ├── src                         // 项目文件
#### │   ├── api                     // 接口类入口
#### │   ├── assets                  // 静态文件夹-存放图片，静态资源等
#### │   ├── axios                   // axios访问拦截配置
#### │   ├── components              // 页面组件
#### │   ├── router                  // 项目路由配置
#### │   ├── store                   // 数据状态管理
#### │   ├── utils                   // 公共组件
#### │   └── views                   // 视图层
### ├── app.vue
### ├── main.js                     
### ├── vue.config.js               //页面配置
